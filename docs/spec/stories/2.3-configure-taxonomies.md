# Story 2.3 — Configure Taxonomies

**Epic**: 2 — Site Configuration **Points**: 2 **Status**: Todo

---

## Story

**As a** blog visitor, **I want** posts organized by tags and categories with browsable taxonomy pages, **So that** I can discover related content by topic.

**Acceptance Criteria**:

- `hugo.toml` taxonomies configured:
  ```toml
  [taxonomies]
    category = "categories"
    tag = "tags"
  ```
- Only `category` and `tag` taxonomies — do NOT include `series` or `author` from theme's example
- `/tags/` page renders without errors (empty until posts exist)
- `/categories/` page renders without errors (empty until posts exist)
- No extra taxonomy pages created (no `/series/`, no `/authors/`)
- Hugo build succeeds with no taxonomy-related warnings

**Testing**: Build and visual verification:

- `hugo build` — no errors or taxonomy warnings
- `hugo server` — `/tags/` and `/categories/` render (empty pages)
- No unexpected taxonomy list pages exist
